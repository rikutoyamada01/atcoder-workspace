# Competitive Programming Workspace 🚀

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Manifest V3](https://img.shields.io/badge/Chrome--Extension-Manifest--V3-blue.svg)](#)
[![AtCoder Compatible](https://img.shields.io/badge/AtCoder-Compatible-brightgreen.svg)](#)

> **AtCoderのページを、エディタ・テスト・提出が統合された「ブラウザ上の開発環境（オンラインIDE）」に変える拡張機能**  
> 環境構築不要、サンプルのコピペ不要、画面・タブ移動不要で、問題画面を開いたその瞬間からコーディングとテストが完結します。

---

## 🌟 コアコンセプト

競技プログラミングを始めようとするとき、多くの初心者が**「高い環境構築の壁（コンパイラの導入やVS Codeの設定）」**や**「エディタ・ブラウザの頻繁な往復（テスト・提出作業）」**で挫折してしまいます。

本拡張機能は、AtCoderの問題ページ内に高性能エディタを直接埋め込むことで、インストールしたその瞬間から**「ブラウザの1タブだけで完結するノンストレスな学習体験」**を提供します。

---

## 🛠️ 主な機能 (MVP)

### 1. 問題文一体型スプリットビューエディタ
- 左側に問題文、右側に高品質なコードエディタ（**Monaco Editor**）を配置。
- 画面サイズに合わせて、ドラッグで分割幅を自在に変更可能。

> [!NOTE]
> 初心者が自力でコーディングする力を養うため、**AI支援やコードの自動入力補完（IntelliSense）はあえて非搭載**としています。ただし、括弧の自動補完や自動インデント、自動コード整形機能により、記述ストレスを最小限に抑えます。

### 2. ワンクリック自動ローカルテスト
- 問題ページ内の「入力例」と「出力例」を自動抽出し、ワンクリックでテスト実行。
- AtCoder公式のカスタムテストAPIを利用して裏でコードを実行し、各テストケースに対して即座に判定（AC/WA）を行います。

### 3. シームレス自動提出 & ポップアップ通知
- エディタ下部の提出ボタンからバックグラウンドで解答コードを提出。
- 提出後は強制的に画面が遷移せず、その場にポップアップで提出結果（提出URL）を表示するため、思考を妨げられません。

### 4. コードの自動保存
- キー入力の停止（1.5秒）またはタブ切り替え時にコードをブラウザのローカルストレージ（`chrome.storage.local`）に自動保存。PCフリーズやタブの誤閉鎖によるコード消失を防ぎます。

### 5. クイックナビゲーション
- エディタ上部に「前の問題へ」「次の問題へ」ボタンを設置。コンテスト一覧画面に戻る手間を省き、テンポよく学習を継続できます。

## 🔄 ユーザー体験の比較

```text
【従来の学習フロー】
VS Codeを開く (環境構築) ➔ ブラウザで問題文を読む ➔ サンプルを手動コピー ➔ ローカルでコンパイル・テスト実行
➔ コードをコピー ➔ 提出タブに貼り付け ➔ 提出後に別画面に強制遷移 (元の問題が見えなくなる)
➔ 一覧画面に戻って次の問題を探す

【本拡張機能のフロー】
AtCoder問題ページを開く (右側にエディタ出現) 
  ↓
コードを書く (自動保存)
  ↓
「テスト実行」ボタンをクリック (自動テスト)
  ↓
「提出」ボタンをクリック (インラインでポップアップ表示)
  ↓
「次の問題へ」をクリック (即座に隣の問題へ遷移)
```

## 🛠️ 技術スタック

* **コア**: HTML5 / JavaScript (ES6+)
* **スタイリング**: CSS3 (レスポンシブデザイン、AtCoderダークテーマ追従型)
* **エディタ基盤**: Monaco Editor (非同期遅延ロード対応)
* **API連携**: AtCoder公式 カスタムテスト・提出エンドポイント
* **データ保存**: Chrome Extension Storage API (`chrome.storage.local`)

## 🚀 開発者向けクイックスタート

ローカル環境で拡張機能を取り込み、開発・デバッグを行う手順です。

### 1. リポジトリのクローン
```bash
git clone https://github.com/yamadarikuto/atcoder-workspace.git
cd atcoder-workspace
```

### 2. Chromeへの読み込み
1. Google Chromeで `chrome://extensions/` を開きます。
2. 右上の「**デベロッパー モード**」を有効にします。
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
4. クローンしたリポジトリのルートディレクトリ（`manifest.json` が配置されているディレクトリ）を選択します。

## 📈 将来的なロードマップ

### フェーズ 2: 学習の定着とエラー分析
* **全ジャッジ結果の自動記録**: ACだけでなく、WA, TLE, RE, CEなどのすべてのエラー・提出結果を保存。
* **エラー傾向の可視化**: 「自分がどの種類のエラー（例：コンパイルエラーやWA）を多く起こしているか」をグラフで分析。
* **学習メモ（Markdown）機能**: 問題ごとに学習メモを書き残せるパネルをエディタと切り替えて表示。

### フェーズ 3: ナレッジベースの構築
* **解法ライブラリ検索**: 過去のACコードやメモをアルゴリズムタグ（二分探索、DPなど）で高速検索。
* **復習キュー**: 過去の間違いが多い問題を最適なタイミングで再提示するスペースド・レペティション機能。
* **Markdown一括エクスポート**: Obsidianなどの第2の脳ツールに連携可能な、YAMLフロントマター付きMarkdownエクスポート機能。
