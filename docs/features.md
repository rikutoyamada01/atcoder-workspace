# 機能仕様詳細

AtCoder Workspaceが提供する各機能の仕様詳細です。

---

## 1. 分割画面エディタ
AtCoderの問題ページを分割し、右側にMonaco Editorを埋め込みます。
* **言語サポート**: C++, Python, Rust, Javaなどのシンタックスハイライト、自動インデント、括弧の自動補完に対応。
* **コード整形**: Prettierによる自動コード整形機能を標準搭載。
* **AIレス設計**: コンテスト中の自力でのコーディング能力を養うため、生成AIによるコード補完やAI支援はあえて非搭載としています。

---

## 2. テンプレートとスニペット
問題を開いた際に、言語別のテンプレートコード（入出力のボイラープレートなど）を自動挿入します。
* **自作ライブラリ管理**: サイドパネルから自分のスニペットやアルゴリズム集をワンクリックで挿入可能。
* **カスタマイズ**: テンプレートの内容はユーザーが自由に編集・追加できます。

---

## 3. ワンクリック自動ローカルテスト
問題文内の入力例・出力例のテキストを自動抽出し、エディタ上のコードに対して一括テストを実行します。
* **完全ローカル実行**: テスト実行はWebAssemblyを用いた仮想環境で処理されるため、AtCoderのサーバーに余計な負荷をかけません。
* **カスタムテスト**: ユーザー自身が検証したい極端な値などのカスタムテストケースを追加登録し、一括実行することができます。

---

## 4. シームレスな自動提出と結果表示
エディタ内の提出ボタンからバックグラウンドで解答コードを送信します。
* **インライン表示**: 提出時に別タブへ強制遷移せず、判定結果（AC, WA, TLE, MLE, RE, CE等）をエディタ内にポップアップで通知します。
* **リアルタイム判定進捗のレンダリング**: WJ (Waiting for Judging) 状態から、ジャッジの進行状況に合わせて「5/12」のように実行中のテストケース数をリアルタイムで表示します。
* **Web Audio API による音声フィードバック**: 判定結果に応じて音によるフィードバックを行います。AC（正解）時は心地よいチャイムのアルペジオ音、WA（不正解等）時は注意を促すダブルビープの警告音が再生されます。
* **デスクトップ通知**: ジャッジ完了時に、Chrome のデスクトップ通知機能を利用して結果をポップアップ通知します。ブラウザのタブが非アクティブな状態であっても即座に結果を把握できます。
* **IndexedDB によるローカルログ保存**: 提出履歴や判定結果のログを、ローカルデータベース（IndexedDB 上の `AtCoderWorkspaceDB` 内 `submissions` ストア）に保存し、過去の提出履歴をオフラインでいつでも参照できます。
* **セッションによるコンソール状態の復元**: ページ遷移や隣接する問題への切り替えを行っても、ブラウザの `sessionStorage` を利用して前回の実行・ジャッジ結果がコンソールに自動復元されます。

---

## 5. コードの自動保存
タイピング停止時（1.5秒経過）や、問題ページのタブ切り替え時に、コードをブラウザのローカルストレージへ自動保存します。
* 不意のブラウザ終了やPCのフリーズによるコード消失を防ぎます。

---

## 6. クイックナビゲーション
エディタ上部から「前の問題」「次の問題」へ直接遷移します。
* コンテスト一覧画面に戻る手間を省き、テンポよく学習を継続できます。

---

## 7. 解答ステータスの個別管理（自力AC / 解説AC）
各問題の正解ステータスを細かく管理し、効果的な復習をサポートします。
* **自力 / 解説の振り分け**: AC時に「自力AC」と「解説AC」を個別に記録します。一度設定されたステータス（解説ACなど）は再提出によって勝手に上書きされることはありません。
* **オプション画面でのステータス一覧**: 拡張機能オプションからこれまでACした問題のステータス一覧を確認でき、手動での変更も可能です。
* **AtCoderへの直接リンク**: オプション画面の一覧表にあるコンテスト名や問題名をクリックすると、新しいタブでAtCoderの該当ページに直接遷移できます（ホバー時のみ下線が表示されるシンプルなデザイン）。
* **柔軟なステータス絞り込み**: オプション画面で「すべて / 自力AC / 解説AC」のステータス別に表示対象を瞬時に絞り込み、復習が必要な問題を簡単に見つけられます。
